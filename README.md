# Buffalos Live

Static GitHub Pages site for BuffalosLive.com.

## Purpose

Minimal black-background Twitch livestream page for the `buffaloslive` Twitch channel.

## Files

- `index.html` — complete production homepage with embedded Twitch player, offline message, and direct Twitch fallback link.
- `logo.png` — Buffalos logo displayed above the player.
- `CNAME` — custom domain configuration for `buffaloslive.com`.

## Hosting

GitHub Pages serves this repository from the `main` branch root.
